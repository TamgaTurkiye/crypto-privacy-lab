# Hashing Passwords

Hashing vs encryption farkı, password hashing best practices, Argon2id/bcrypt/PBKDF2, salt/pepper kavramları.

## Scope

Bu klasör savunma, güvenli kullanım ve ürün tasarımı odaklı içerikler içindir.

## Out of Scope

- Gerçek secret/key/token/password yok
- Hash cracking, password list veya credential abuse yok
- Exploit/PoC veya aktif saldırı akışı yok
- Kendi kripto algoritmanı yazma yönlendirmesi yok
