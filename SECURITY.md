# Security Policy

Bu repo savunma odaklı araştırma ve dokümantasyon deposudur.

Lütfen gerçek secret, token, password, private key, credential, leaked data, exploit, cracking workflow veya aktif saldırı adımı içeren hiçbir içeriği issue/PR olarak göndermeyin.

Genel güvenlik politikası:
https://github.com/TamgaTurkiye/.github/blob/main/SECURITY.md

## Reporting a Security Concern

Bir güvenlik endişesi fark ederseniz, public issue açmadan önce repository maintainer ile özel kanaldan iletişime geçin.

## Out of Scope

- Gerçek credential veya key paylaşımı
- Hash cracking workflow
- Password list
- Credential abuse
- Exploit/PoC
- Aktif saldırı veya kırma talimatları
